# Logo Files

Place your logo files in this directory:

- `light.png` - Logo for light backgrounds (white/light gray logo)
- `dark.png` - Logo for dark backgrounds (dark logo)

## Recommended Specifications:

- **Format**: PNG with transparent background
- **Size**: 512x512px or larger (will be scaled down)
- **Aspect Ratio**: Square or horizontal rectangle
- **File Size**: Under 100KB for optimal loading

## Usage:

These logos are used in:
- Header navigation
- Footer branding
- Brand assets page
- Social media cards (Open Graph)

If logo files are missing, the ThemeAwareLogo component will show a placeholder.
